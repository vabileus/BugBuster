## Simple bug tracker

React, Redux, Nestjs, MongoDB (Atlas)

### Table of contents

* [Install](#install)
* [Usage](#usage)
* [License](#license)

### Install

#### Clone this repo

```
git clone https://github.com/vabileus/BugBuster.git
```

#### Install dependencies

```
npm install
```

### Usage

#### Run the client part

```
cd app
```

```
npm run start
```

#### Run the server part

```
cd api
```

```
npm run start
```

#### Build the app (manual)

```
npm run build
```
